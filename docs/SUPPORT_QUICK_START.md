## ğŸ“ Common Tasks

### Create a Support Ticket (30 seconds)

1. Go to `/support`
2. Click **"Create New Support Ticket"**
3. Fill in the form:
   - **Subject**: Brief description (e.g., "Cannot complete payment")
   - **Category**: Select issue type
   - **Priority**: How urgent is it?
   - **Description**: Detailed explanation
4. Click **"Create Ticket"**
5. Note your ticket number (e.g., TKT-20251024-0001)

**Ticket will be:**
- Automatically assigned a unique number
- Tracked through status changes
- Responded to based on priority

### Start Live Chat (10 seconds)

1. Go to `/support`
2. Click **"Live Chat"** tab
3. Click **"Start Chat"**
4. Type your question
5. Get instant AI response

**Chat Features:**
- Available 24/7
- Supports English & Swahili
- Remembers conversation history
- Can escalate to human agent

### Search Help Articles (5 seconds)

1. Go to `/support`
2. Click **"Help Center"** tab
3. Type keywords in search box
4. Browse results
5. Click on article to read
6. Vote if it was helpful

### Submit Feedback (20 seconds)

1. Go to `/feedback`
2. Click **"Give Feedback"**
3. Rate with stars (1-5)
4. Select category
5. Add comment (optional)
6. Click **"Submit Feedback"**

**Your feedback:**
- Helps us improve
- Is analyzed for sentiment
- Contributes to statistics
- Remains anonymous if preferred

### Change Language (3 seconds)

1. Look for language selector (top-right)
2. Click to open menu
3. Select **English** or **Kiswahili**
4. Page updates instantly
